# Question
After each of the calls to write() in the following code, explain what the content of
the output file would be, and why:
fd1 = open(file, O_RDWR | O_CREAT | O_TRUNC, S_IRUSR | S_IWUSR);
fd2 = dup(fd1);
fd3 = open(file, O_RDWR);
write(fd1, "Hello,", 6);
write(fd2, "world", 6);
lseek(fd2, 0, SEEK_SET);
write(fd1, "HELLO,", 6);
write(fd3, "Gidday", 6);

# Answer
- write 1:
offset(fd1) = 0 -> write 6 bytes "Hello," -> offset(fd1) = 6
=> output: Hello,

- write 2:
offset(fd1) = offset(fd2) = 6
-> write 6 bytes of "world"
=> output: Hello,world

- write 3:
use lseek(fd2, 0, SEEK_SET) => offset(fd1) = offset(fd2) = 0
-> write 6 bytes of "HELLO," will overwrite byte from offset 0 to 5
=> output: HELLO,world

- write 4:
offset(fd3) = 0 because it's just opened.
=> write 6 bytes of "Gidday" will overwirte byte from offset 0 to 5 too.
=> output: Gidday,world
