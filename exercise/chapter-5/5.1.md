# Question
5.1. Modify the program in Listing 5-3 to use standard file I/O system calls (open() and lseek()) and the off_t data type. Compile the program with the _FILE_OFFSET_BITS
macro set to 64, and test it to show that a large file can be successfully created.

# Answer
The code already has in <root_folder>/fileio/large_file.c, so we just need to modify them into the function in 32-bit mode we already know. Then just compile them with additional argument in gcc, like `gcc -D_FILE_OFFSET_BITS=64 <rest of command>`
